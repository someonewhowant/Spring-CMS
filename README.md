# Spring Blog Platform

![Banner](readme-image.jpg)

Project Nova is a radical, celestial-themed blog platform built with **Spring Boot 3**, **Thymeleaf**, and a high-performance design system. It is engineered for those who reach beyond the conventional, combining a deep-space aesthetic with robust enterprise technology.

## ✨ Features

- **🚀 High Performance**: Built on Spring Boot 3 for rapid response times and scalability.
- **🎨 Modern Aesthetics**: A clean, "glassmorphic" interface using **Plus Jakarta Sans** and a vibrant Indigo/Emerald color palette.
- **🌓 Adaptive Theme**: Fully supports Light and Dark modes based on system preferences.
- **🔍 Full-Text Search**: Integrated search functionality to find insights across all publications.
- **🛠️ Admin Dashboard**: A comprehensive, secure administrative portal for managing posts, users, and content.
- **📱 Responsive Design**: Optimized for all devices, from desktop monitors to mobile phones.
- **📝 Markdown Support**: (Optional/Concept) Write your posts with simplicity and speed.

## 🛠️ Tech Stack

- **Backend**: Java 17, Spring Boot 3.x, Spring Data JPA, Spring Security
- **Database**: H2 (In-memory for development)
- **Frontend**: Thymeleaf, HTML5, Vanilla CSS3 (Custom Design System), JavaScript (ES6+)
- **Icons**: Bootstrap Icons

## 🚀 Getting Started

### Prerequisites

- Java 17 or higher
- Maven 3.x

### Installation

1. **Clone the repository**:
   ```bash
   git clone https://github.com/your-repo/spring-blog.git
   cd spring-blog
   ```

2. **Build the project**:
   ```bash
   mvn clean install
   ```

3. **Run the application**:
   ```bash
   mvn spring-boot:run
   ```

The application will be available at `http://localhost:8080`.

## 📂 Project Structure

- `src/main/java`: Core logic and backend architecture.
- `src/main/resources/templates`: Thymeleaf templates with layout fragmentation.
- `src/main/resources/static`: Custom design system (CSS), JS, and assets.
